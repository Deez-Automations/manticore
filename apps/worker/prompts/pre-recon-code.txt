## STEP 1 — Map the codebase (do all of these before reading any files)

Run these in sequence:
1. `Glob("**/*.{php,py,js,ts,rb,go,java}")` — find all application source files
2. `Glob("**/*.{json,yaml,yml,xml,conf,ini,env,htaccess}")` — find config and schema files
3. `Read(".gitignore")` if it exists — note excluded paths
4. `Bash("find {{REPO_PATH}} -maxdepth 2 -type d | head -60")` — understand top-level directory structure

From the results, identify the entry point files (index.php, app.py, main.go, server.js, etc.) and the config directory. Do NOT read every file yet.

---

## STEP 2 — Read core application files (in this order, skip external/vendor)

Read these files one at a time:
1. The main entry point (index.php / app.py / main.go / etc.)
2. Authentication files: login, logout, signup, password reset handlers
3. Session/cookie configuration files
4. Main config files (database config, app config)
5. Request routing file (routes.php, urls.py, router.go, etc.)
6. Core middleware files (auth middleware, input validation middleware)
7. Database access layer (models, repositories, ORM config)
8. File upload handlers if present
9. Any admin panel entry points

For each file read, note: what user input it accepts, how it validates, what it queries/executes, what it outputs.

## STEP 3 — Run security-focused Grep searches

Run each of these Grep searches and read the top results:

**XSS sinks:**
- `Grep("innerHTML|document\.write|eval\(|setTimeout\(|setInterval\(", type="js")`
- `Grep("echo\s+\$_(GET|POST|REQUEST|COOKIE)|print\s+\$_(GET|POST)", type="php")`

**SQL Injection:**
- `Grep("\$_(GET|POST|REQUEST|COOKIE).*query|query.*\$_(GET|POST|REQUEST|COOKIE)", type="php")`
- `Grep("SELECT|INSERT|UPDATE|DELETE.*\$_(GET|POST|REQUEST)", type="php")`

**Command Injection:**
- `Grep("system\(|exec\(|shell_exec\(|passthru\(|popen\(|proc_open\(", type="php")`

**File Inclusion / Path Traversal / SSRF:**
- `Grep("include\(\s*\$_(GET|POST)|require\(\s*\$_(GET|POST)|file_get_contents\(\s*\$_(GET|POST)|fopen\(\s*\$_(GET|POST)", type="php")`
- `Grep("curl_setopt|curl_exec|file_get_contents.*http", type="php")`

**Authentication & Session:**
- `Grep("session_start|session_destroy|session_regenerate_id|setcookie", type="php")`
- `Grep("password_verify|md5\(|sha1\(|hash\(", type="php")`

**CSRF:**
- `Grep("csrf|token.*form|form.*token", output_mode="files_with_matches")`

For each result set, read the most interesting files to understand context.
